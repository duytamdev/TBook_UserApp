# TBook_UserApp


## Giới thiệu
- Một ứng dụng giúp bạn dể dàng mua những quyển sách yêu thích.

## Các tính năng
- Đăng kí , đăng nhập
- Thêm sản phẩm vào giỏ hàng
- Yêu thích 
## Công nghệ sự dụng
  - React native CLI
  - Kết nối với backend bằng API
  - <a target="_blank" href="https://github.com/duytamdev/TBook">Link backed <a/>

- Reactnavigation
- Redux
- Lottie
- ...
